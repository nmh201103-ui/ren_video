# Link sản phẩm TikTok Shop — dùng link gì, kiểm tra thế nào

## Trên điện thoại bạn "thêm vào giỏ rồi tạo video" — tool làm thế nào?

Trên **điện thoại** bạn hay: thêm sản phẩm vào giỏ → rồi bấm **Tạo video với sản phẩm đó**. Trên **laptop + emulator** làm tương tự, nhưng chia hai phần:

1. **Tool (trên laptop)** chỉ làm:
   - Render video từ Shopee/sản phẩm.
   - **Đẩy file video** vào emulator (nút "Đẩy video lên TikTok").
   - Mở app TikTok trong emulator.

2. **Trong app TikTok (trên emulator)** bạn làm giống điện thoại:
   - Vào **TikTok Shop** → tìm sản phẩm cần đẩy → **thêm vào giỏ** (hoặc "Lưu"/"Sản phẩm của tôi" tùy phiên bản app).
   - Bấm **+** (Tạo) → **Tải lên** → chọn video vừa được tool đẩy vào (trong Thư viện/Download).
   - Khi đăng video: bấm **Thêm link / Sản phẩm** (hoặc "Tạo video với sản phẩm") → **chọn sản phẩm từ giỏ** (hoặc danh sách sản phẩm đã lưu) → Đăng.

**Tóm lại:** Tool không thay bạn "thêm vào giỏ" — việc đó bạn làm trong app TikTok trên emulator (giống điện thoại). Tool chỉ lo: tạo file video + đẩy file vào emulator + mở TikTok; bước chọn sản phẩm khi đăng bạn làm trong app.

---

## Dùng link gì? (để lưu lại / kiểm tra)

- **Dùng link trang sản phẩm TikTok Shop**, không cần link "giỏ hàng".
- Khi xem video bấm vào link → mở **trang sản phẩm** → họ tự bấm "Thêm vào giỏ" nếu muốn mua.
- Bạn **không cần** biết link "đã thêm vào giỏ hàng chưa" — chỉ cần link **đúng sản phẩm** (affiliate của bạn).

## Lấy link ở đâu?

1. Mở **TikTok** (app hoặc web) → vào **TikTok Shop**.
2. Tìm **sản phẩm** bạn đang làm affiliate.
3. Mở **trang sản phẩm** đó.
4. Bấm **Chia sẻ** (Share) → **Sao chép link** (Copy link).
5. Dán link đó vào ô **Link sản phẩm TikTok Shop** trong tool.

## Làm sao biết link đúng (đã đúng sản phẩm)?

1. Dán link vào ô trong tool.
2. Bấm **Mở link** → trình duyệt mở link.
3. Nếu thấy **đúng trang sản phẩm** (tên, giá, ảnh đúng) → link đúng, dùng khi đăng video.
4. Nếu mở ra trang khác hoặc lỗi → đổi link (lấy lại từ TikTok Shop).

**Tóm lại:** Dùng **link sản phẩm TikTok Shop** (Copy từ Chia sẻ). Kiểm tra bằng nút **Mở link**. Không cần link giỏ hàng hay kiểm tra "đã thêm vào giỏ chưa".
